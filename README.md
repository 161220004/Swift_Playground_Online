### 部署

- 打开 Docker Desktop

- 终端进入目录：Swift_Playground_Online

- 命令行输入：

  - 创建SPO项目镜像“aldebarain/spo:latest”

    `docker-compose build`  

  - 创建该镜像的容器并运行

    `docker-compose up`
